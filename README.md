# KUA Waiting List App

## Project Structure
- `backend/`: Firebase Functions
- `frontend/`: Flutter Application
- `docker/`: Docker configuration
- `.github/`: CI/CD workflows

## Setup
1. Install Flutter SDK and Firebase CLI.
2. Run `make setup` to initialize the environment.
3. Navigate to `frontend` and run `flutter pub get`.
4. Navigate to `backend/functions` and run `npm install`.

## Development
Run `make dev` to start the development server.
